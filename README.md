# Neurable MW75 Neuro — Steam Deck EEG Pipeline

Real-time 12-channel EEG acquisition from Neurable MW75 Neuro headphones on Steam Deck (Arch Linux).

## Hardware
- **Headphones:** Neurable MW75 Neuro (12-channel dry EEG)
- **Host:** Steam Deck (Arch Linux + bluetooth/GAIA BLE)
- **Target:** Jetson Orin Nano (Eteru-Orion) via network relay
